# Page-Replacement-Simulator
A web-based simulator for page replacement algorithms (FIFO, LRU, Optimal) with step-by-step visualization, frame matrix display, and Belady's Anomaly detection.

## Features
- FIFO, LRU, and Optimal algorithms
- Step-by-step playback with animated frame matrix
- Compare all algorithms side by side
- Belady's Anomaly detector with frame sweep
- Random page string generator
- Export results to text file

## How to Run
Just open `page_replacement_simulator.html` in any browser. No installation needed.
